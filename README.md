A 3D basketball game with full General Manager mode. Build your dynasty!

## Features

### Phase 1: 3D Gameplay
- Full 3D basketball court with Three.js
- Player movement and controls
- Shooting with power meter
- Passing between teammates
- Ball physics (bouncing, scoring detection)
- AI opponents

### Phase 2: GM Architecture  
- Player roster management
- Salary cap system
- Free agency signings
- Player stats and ratings

### Phase 3: Full Manager Mode
- 30-team league simulation
- Season schedule and standings
- Advance weeks to simulate games
- Player development over time
- Financial management (budget, luxury tax)

## Controls

| Key | Action |
|-----|--------|
| W/A/S/D | Move player |
| Space | Shoot (hold for power) |
| E | Pass to teammate |
| Q | Switch controlled player |

## Tech Stack

- **Vite** - Build tool
- **TypeScript** - Type safety
- **Three.js** - 3D rendering
- **Vanilla JS** - No framework bloat

## Development

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run build
```

Deploy the `dist` folder to any static host.
